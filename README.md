# Excel-SQL-SuperShop-Sales-Analysis

## Overview
Interactive sales analytics dashboard built using SQL and Microsoft Excel to analyze sales, profit, orders, category performance, regional trends, and key business KPIs.

## Tools Used
- SQL
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## What I Did
- Analyzed sales data using SQL.
- Cleaned and prepared the dataset.
- Calculated key KPIs (Sales, Profit, Orders, Quantity, Profit Margin, YoY Growth).
- Built an interactive Excel dashboard with Pivot Tables, Charts, Slicers, and Maps.
- Visualized category, regional, and monthly sales performance.
